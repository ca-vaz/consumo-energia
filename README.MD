💡 Consumo de Energia - Simulador Mensal

📌 Sobre o Projeto
Este script em Python foi desenvolvido para facilitar o cálculo do consumo elétrico residencial. Ele permite que o usuário identifique o impacto de equipamentos específicos no orçamento mensal, promovendo uma maior consciência sobre o uso de energia.

🧮 Fórmulas Utilizadas

O cálculo baseia-se nas seguintes equações matemáticas:

Consumo Mensal (kWh):$$Consumo = \frac{Potência (W) \times Horas/Dia \times 30}{1000}

$$Custo Mensal (R$):$$Custo = Consumo \times Tarifa$$

(Neste projeto, a tarifa padrão aplicada é de R$ 0,75/kWh).

🚀 Como Utilizar
Para rodar o simulador, siga os passos abaixo:

1. Execute o arquivo Python.
2. Insira o nome do aparelho (ex: Geladeira).
3. Informe a potência do dispositivo em Watts (W).
4. Informe a média de tempo de uso diário em horas (h).

O programa retornará automaticamente o consumo total em kWh e o valor estimado em Reais.
