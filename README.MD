# Consumo de energia

##Objetivo:
Este projeto em Python foi desenvolvido para calcular o consumo aproximado de energia que um determinado equipamento terá no mês. Além disso, será possível determinar o valor, em reais, que este dispositivo irá somar na conta de luz mensal. 

As fórmulas utilizadas para os cálculos de consumo e valor de energia estão, respectivamente, descritas a seguir:
**consumo_mensal** = (potencia * tempo_horas * 30) / 1000
**valor_consumo_mensal** = (consumo_mensal * 0.75)

##Instruções para utilização:
Para utilizar o programa, basta entrar com o **nome do aparelho**, a sua **potência** (em Watts) e o tempo pelo qual ele é usado em um dia(em horas). Essas informações resultarão no cálculo do consumo mensal e o valor de energia a ser pago na conta de luz. 

💡 Consumo de Energia - Simulador Mensal

📌 Sobre o Projeto
Este script em Python foi desenvolvido para facilitar o cálculo do consumo elétrico residencial. Ele permite que o usuário identifique o impacto de equipamentos específicos no orçamento mensal, promovendo uma maior consciência sobre o uso de energia.

🧮 Fórmulas UtilizadasO cálculo baseia-se nas seguintes equações matemáticas:

Consumo Mensal (kWh):$$Consumo = \frac{Potência (W) \times Horas/Dia \times 30}{1000}

$$Custo Mensal (R$):$$Custo = Consumo \times Tarifa$$

(Neste projeto, a tarifa padrão aplicada é de R$ 0,75/kWh).

🚀 Como Utilizar
Para rodar o simulador, siga os passos abaixo:
Execute o arquivo Python.
Insira o nome do aparelho (ex: Geladeira).
Informe a potência do dispositivo em Watts (W).
Informe a média de tempo de uso diário em horas (h).
O programa retornará automaticamente o consumo total em kWh e o valor estimado em Reais.
